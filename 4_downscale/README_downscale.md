# Downscaling

Downscaling scripts run on the Bristol HPC. The code is maintained separately on the server.

This folder is a placeholder — copy downscaling scripts here before running.

## Overview

Post-emulator step that bias-corrects and downscales the 3.75° × 2.5° HadCM3 grid predictions to higher-resolution site data for the 112 UK repository sites.

## Running on HPC

```bash
# Copy scripts to HPC, then:
sbatch downscale_job.sh
```
